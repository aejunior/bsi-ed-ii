## Instanciando um ambiente virtual com Python3
```
$ python3 -m venv venv
```

## Ativando o ambiente virtual
UNIX
```
$ source venv/bin/activate
```

Windows
```
$ venv\Scripts\activate.bat # Prompt
$ venv\Scripts\activate.ps1 # PowerShell
```

## Executando o projeto
```
$ python3 main.py
```
